# Meu Look — Deploy no GitHub Pages

## 📋 O que você tem aqui

- `index.html` — seu app completo com PWA
- `manifest.json` — configuração para virar um "app"
- `sw.js` — permite funcionar offline
- Este arquivo `README.md`

## 🚀 Passo a passo para colocar no ar

### 1️⃣ Criar repositório no GitHub

1. Abra **github.com**
2. Clique no **+** (canto superior direito) → **New repository**
3. **Nome do repositório:** `seu-username.github.io`
   - Exemplo: se seu username é "caroline_santos", fica `caroline_santos.github.io`
4. Deixa **Public** (marcado)
5. Clica **Create repository**

### 2️⃣ Upload dos arquivos (forma fácil)

1. Na página do repositório que acabou de criar, clique em **Add file** → **Upload files**
2. Arraste estes 4 arquivos para a janela:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `README.md`
3. Clique em **Commit changes** (deixa o texto padrão)
4. **Pronto!** Espera 1-2 minutos e seu app está em: `https://seu-username.github.io`

### 3️⃣ Instalar no iPhone

1. Abra **Safari**
2. Va em: `https://seu-username.github.io`
3. Clique no botão **Compartilhar** (quadradinho com seta)
4. Selecione **"Adicionar à Tela de Início"**
5. Dê um nome (ex: "Meu Look") e clique **Adicionar**

**Pronto!** O app fica na sua home como um app de verdade 📱

## 🔄 Como atualizar depois

Toda vez que você quer fazer uma alteração:

1. Va no GitHub (`seu-username.github.io`)
2. Clique em um arquivo
3. Clique no lápis (✏️) para editar
4. Faça suas mudanças
5. Clique **Commit changes**
6. No iPhone, abra o app e faça um **pull down** (puxar a tela pra baixo)
7. **Pronto**, tá atualizado!

## 📝 Dúvidas comuns

**"Pode alguém encontrar meu app?"**
- Não. A menos que você compartilhe o link, ninguém descobrir aleatoriamente.

**"Meus looks/peças ficam salvos?"**
- Sim! Tudo fica no localStorage do seu iPhone (local, na sua mão).

**"Funciona offline?"**
- Sim! Depois da primeira vez que você abre, funciona até sem internet.

**"Preciso de alguma assinatura?"**
- Não, GitHub Pages é grátis.
